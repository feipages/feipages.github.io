## ionic 入门

### 1. 通过npm搭建开发环境
``` 
$ sudo npm install -g ionic@4.11.10
$ sudo npm i @ionic/core@4.11.10
sudo npm i @ionic/react@4.11.10
sudo npm i @ionic/angular@4.11.10

``` 
### 2. 初始化项目
```
# npm 6 以上的版本
$ npm init react-app my-app
#低版本的npm
$ npx create-react-app my-app

```
