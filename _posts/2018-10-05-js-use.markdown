1. Web JS的组成
ECMAScript + DOM + BOM

ECMAScript（前身为欧洲计算机制造商协会）
JavaScript的语法规范
DOM（Document Object Model 的简称）
JavaScript操作网页上元素的API
BOM（Browser Object Model 的简称）
JavaScript操作浏览器部分功能的API
