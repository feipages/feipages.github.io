## angular 入门

### 1. angular
特点

``` 
# 安装ionic 命令行工具
$ sudo npm install -g @angular/cli



``` 
### 2. 初始化项目
```
cd 到需要创建项目的目录下
$ ng new <projectname>
# 1. 先创建，再安装
$ ng new <projectname> --skip-install
$ cd <projectname>
$ cnpm install

#运行
$ ng serve --open

#创建组件,在app目录下，创建components目录并创建home组件
$ ng g component components/home

```
