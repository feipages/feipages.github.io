# maven 介绍

maven 安装
1. 下载 maven
2. 解压

```
mv apache-maven-3.5.4 /usr/local/maven3
```

```
# maven home
MAVEN_HOME=/usr/local/maven3
export MAVEN_HOME
export PATH=$PATH:$MAVEN_HOME/bin

```
